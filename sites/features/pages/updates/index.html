Updates that don't eat your data.

The problem

OS updates are huge. On expensive data, people skip them. Security suffers.

The solution

Circle OS uses delta updates (only what changed) and mesh distribution (download from nearby devices). Updates are 90% smaller.

How it works

When an update is available, Circle OS checks if nearby devices have it. If so, download locally for free. Otherwise, download just the changes.

Benefits

  • 90% smaller
  • Mesh sharing
  • WiFi-only option
  • Seamless install
Technical details

Delta updates (bsdiff). A/B partitions (seamless). SleptOn OTA. Mesh chunk distribution.
