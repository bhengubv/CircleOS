# Circle OS — Fact Sheet

## Overview

Circle OS is a privacy-first mobile operating system built on Android 14 (AOSP). It gives old phones a second life with no Google services, offline-first mesh networking, on-device AI, and tap-to-pay without internet.

## Key features

| Feature | What it does |
|---------|-------------|
| **Privacy Engine** | Default-deny internet, identifier spoofing, permission auto-revoke |
| **Butler AI** | On-device AI assistant — no cloud, no subscription |
| **Titanium Wallet** | NFC tap-to-pay without internet (Shongololo currency) |
| **Mesh Network** | Phone-to-phone messaging when internet is down |
| **File DMZ** | Every inbound file sanitised before access |
| **Dual Boot** | Keep Android for banking; Circle OS for everything else |
| **SleptOn** | Privacy-first app store with mesh distribution |
| **Delta OTA** | ~90% smaller updates, shareable via mesh |
